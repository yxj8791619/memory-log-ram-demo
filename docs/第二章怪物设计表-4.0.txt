《Project: Memory.log》第二章怪物设计表 4.0

文档定位：
本文件基于《第二章总设计文档 4.0》与《第二章六种基础动作表 4.0》编写，
用于确定第二章敌人生态、层级分工、战斗职责与教学顺序。

本文件不继承旧版怪物草案。
所有怪物都以“是否服务双层决策”和“是否能与六种基础动作形成稳定互动”为第一判断标准。

--------------------------------------------------
一、总设计原则
--------------------------------------------------

第二章怪物设计不追求单只怪概念很炫，而追求三件事：

1. 玩家一眼能感受到“这怪在哪层更该处理”
2. 玩家在 B 层时，能清楚判断“该稳杀，还是该切回 A 层”
3. 不同怪物能分别验证六种基础动作的切层职责，而不是都被同一种最优手段吃掉

每只怪都必须回答五个问题：
1. 它在哪一层更强
2. 它在哪一层更适合被处理
3. 它是否鼓励被 B -> A 切回
4. 它主要被哪类动作克制
5. 它在混战中的职责是什么

--------------------------------------------------
二、敌人生态总览
--------------------------------------------------

第二章基础敌人分为五类：

1. A 层优势怪
玩家应尽量把它送回 A 层处理，或者避免在 B 层长时间纠缠。

2. B 层优势怪
玩家应主动创造机会在 B 层稳妥击杀。

3. 可转场怪
两层都能打，但是否切回 A 层，会显著改变战局节奏。

4. 功能怪 / 支援怪
它们通过护盾、干扰、封锁、标记等方式改变其它怪的层级解法。

5. 节奏怪 / 场控怪
它们不一定单体最强，但负责打断玩家节奏、压缩站位、逼出切层。

--------------------------------------------------
三、怪物详细设计
--------------------------------------------------

【E-01】前台猎犬 Front Hound

分类：
A 层优势怪

外观与感觉：
瘦长、高速、像正在过度索引的搜索进程。身体边缘不断闪烁出拖尾。

A 层表现：
速度快，但运动规则清晰，容易被枪风筝和点射。
会短距离扑击，但前摇明显。

B 层表现：
在低重力和慢节奏环境下，它的扑击轨迹变得更飘、更难读，且容易绕到玩家头顶。

更适合处理的层：
A 层

玩家标准解法：
1. 如果它本来就在 A 层，直接枪压制和走位处理
2. 如果战斗中跟着玩家进了 B 层，优先用“锤攻击 + 切层键”把它带回 A 层

是否鼓励 B 层稳杀：
不鼓励

是否鼓励 B -> A 切回：
强烈鼓励

主要对接动作：
- B 层组合切技
- A 层普通攻击

设计价值：
这是最直白的“有些怪不该在 B 层硬打”的教学怪。

--------------------------------------------------

【E-02】页锁盾卫 Page-Lock Guard

分类：
B 层优势怪

外观与感觉：
厚重、方块化、像被系统强行锁页的守卫进程，前方展开大面积板状护盾。

A 层表现：
正面极耐打，会缓慢推进压迫玩家。
枪械伤害显著衰减，逼迫玩家绕背或转层。

B 层表现：
护盾变重、转身更慢、背后暴露更多，容易被重击和近战破甲。

更适合处理的层：
B 层

玩家标准解法：
1. A 层先建立位置与资源
2. 主动切入 B 层，绕背或压重击处理
3. 如果没把握一波打死，不建议过早切回 A 层

是否鼓励 B 层稳杀：
强烈鼓励

是否鼓励 B -> A 切回：
谨慎，不作为默认打法

主要对接动作：
- A 层控制切层护盾
- B 层重击
- B 层重击切层

设计价值：
这是第二章“错误层刮痧，正确层兑现”的核心样板怪。

--------------------------------------------------

【E-03】碎片蜂群 Fragment Swarm

分类：
节奏怪 / 场控怪

外观与感觉：
由一串小型数据碎片组成的飞虫群，像坏掉的 UI 粒子在乱飞。

A 层表现：
会不断横向骚扰、扰乱瞄准，但单体很脆。

B 层表现：
会在低重力中聚成团，单体更慢，但容易形成包围。

更适合处理的层：
视数量而定

玩家标准解法：
1. 少量时在 A 层点掉
2. 成群时优先考虑用 B 层重击切层把群体带回 A 层，然后接 AOE 收掉

是否鼓励 B 层稳杀：
中等，不是最优

是否鼓励 B -> A 切回：
强烈鼓励，尤其对群体

主要对接动作：
- A 层炸弹切层
- B 层重击切层

设计价值：
这是群体带回 / AOE 改场的主要测试怪。

--------------------------------------------------

【E-04】后台炮台 Background Turret

分类：
B 层优势怪 / 功能怪

外观与感觉：
固定在空间节点上的悬浮写入器，像一座不断向前台吐出垃圾数据的后台设备。

A 层表现：
能跨维度朝 A 层喷射数据弹，给玩家持续压力。
自身不一定完全无敌，但在 A 层极难安全输出。

B 层表现：
本体完整暴露，机动性差，适合切入点杀。

更适合处理的层：
B 层

玩家标准解法：
先在 A 层躲弹幕、建立资源，再切入 B 层快速处理。

是否鼓励 B 层稳杀：
强烈鼓励

是否鼓励 B -> A 切回：
一般，不是重点

主要对接动作：
- A 层普通攻击切层风道
- A 层控制切层护盾
- B 层普通锤击

设计价值：
它负责建立“B 层是进攻处理层，不只是连招层”的认知。

--------------------------------------------------

【E-05】换页祭司 Swap Priest

分类：
功能怪 / 支援怪

外观与感觉：
漂浮、瘦高、不断放出几何环形符号，像在重写敌人权限表。

A 层表现：
会给周围敌人附加“前台稳态”效果。
效果例子：
- 提高 A 层抗性
- 缩短被切层后暴露时间
- 提升冲刺和扑击频率

B 层表现：
本体更脆，但会主动远离玩家并尝试躲在其它怪后面。

更适合处理的层：
通常先去 B 层处理

玩家标准解法：
先判断它是否在给核心敌人上增益。
若是，优先切层处理它，再回头打主怪。

是否鼓励 B 层稳杀：
高

是否鼓励 B -> A 切回：
中等

主要对接动作：
- B 层组合切技
- B 层控制技切层抢位

设计价值：
它让玩家学会“不是所有怪都先打血最厚的，层级支援目标要先处理”。

--------------------------------------------------

【E-06】回收执行者 GC Enforcer

分类：
可转场怪

外观与感觉：
体型中大型，双臂像回收夹，移动时伴随方形扫描框。

A 层表现：
攻击节奏慢，但范围大，容易封路线。
在 A 层比较好读招，也更容易被枪蹭血。

B 层表现：
动作更连贯、压迫更强，但被重击打断时硬直更大。

更适合处理的层：
两层都能打，但打法不同

玩家标准解法：
1. 想稳妥时，在 A 层拉扯消耗
2. 想打高收益时，在 B 层压到残血后决定是稳杀还是切回 A 层收尾

是否鼓励 B 层稳杀：
高

是否鼓励 B -> A 切回：
高

主要对接动作：
- B 层组合切技
- B 层重击切层
- A 层收尾火力

设计价值：
这是“B 层稳杀”和“B -> A 转场收尾”分流的核心样板怪。

--------------------------------------------------

【E-07】锚点潜伏者 Anchor Lurker

分类：
A 层优势怪 / 节奏怪

外观与感觉：
像藏在地形边缘的数据钩爪，会突然从节点位置冲出。

A 层表现：
会从固定锚点出现，但规律可记忆，适合前台走位处理。

B 层表现：
锚点可见性变差，且会利用低重力和高低差追击玩家，容易打断慢动作。

更适合处理的层：
A 层

玩家标准解法：
尽量别在 B 层和它拖。
若已经在 B 层接敌，应尽快用“锤攻击 + 切层键”送回 A 层。

是否鼓励 B 层稳杀：
低

是否鼓励 B -> A 切回：
强烈鼓励

主要对接动作：
- B 层组合切技
- B 层控制技切层位移

设计价值：
它测试玩家是否会用“切回更有利层”而不是在错误层硬扛。

--------------------------------------------------

【E-08】扇区布雷者 Sector Miner

分类：
节奏怪 / 场控怪

外观与感觉：
不断投下小型数据包，包体落地后延迟爆开。

A 层表现：
布雷密度一般，但地图压迫强，适合用远程点掉本体。

B 层表现：
雷包会变成更慢但更持久的悬浮危险区。
本体移动更慢，容易被近身重击。

更适合处理的层：
本体更适合在 B 层处理，雷区更需要跨层规避

玩家标准解法：
1. 先看场面是 A 雷压迫还是 B 雷压迫
2. 利用 A 层炸弹切层、B 层重击切层重排敌群和危险区位置

是否鼓励 B 层稳杀：
中高

是否鼓励 B -> A 切回：
看场面，有收益但非必选

主要对接动作：
- A 层炸弹切层
- B 层重击切层

设计价值：
它是“敌人与场控一起构成切层决策”的代表。

--------------------------------------------------

【E-09】权限吸附体 Privilege Leech

分类：
功能怪 / 支援怪

外观与感觉：
像一团附着在别人身上的代码寄生体，会发出不稳定脉冲。

A 层表现：
附着到其它怪身上后，增强该目标的 A 层攻击性或抗性。

B 层表现：
附着关系会短暂断开，本体暴露且脆弱，但逃跑很快。

更适合处理的层：
B 层优先秒掉

玩家标准解法：
遇到它就要迅速判断是否值得专门切层点杀。

是否鼓励 B 层稳杀：
高

是否鼓励 B -> A 切回：
低，中断它比带回更重要

主要对接动作：
- B 层普通锤击
- B 层控制技切层位移追猎

设计价值：
它给 B 层控制技一个明确的追击目标和功能价值。

--------------------------------------------------

【E-10】缓存巨像 Cache Colossus

分类：
大型 B 层优势怪 / 小型精英

外观与感觉：
由很多漂浮缓存块拼成的大型体，动作沉重但极具压迫感。

A 层表现：
可被枪消耗，但效率极低，且会周期性制造大面积前台冲击。

B 层表现：
外层块体可被重击打散，露出短时弱点。
是典型的“进 B 层兑现爆发”的对象。

更适合处理的层：
B 层

玩家标准解法：
1. A 层积攒资源和控节奏
2. B 层重击拆甲
3. 看局势决定是在 B 层稳杀，还是用高成本切层带回 A 层打大收尾

是否鼓励 B 层稳杀：
高

是否鼓励 B -> A 切回：
中高，但应建立在已拆甲和血线健康的前提上

主要对接动作：
- A 层控制切层护盾
- B 层重击
- B 层重击切层

设计价值：
它是第二章中后段的大型节奏检验怪。

--------------------------------------------------
四、敌人分层用途表
--------------------------------------------------

强烈建议送回 A 层处理的怪：
- Front Hound
- Anchor Lurker

强烈建议在 B 层稳杀的怪：
- Page-Lock Guard
- Background Turret
- Cache Colossus

适合在 B 层打到一半，再决定是否切回 A 层的怪：
- GC Enforcer
- Fragment Swarm
- Sector Miner

优先作为功能目标处理的怪：
- Swap Priest
- Privilege Leech

--------------------------------------------------
五、与六种基础动作的对接表
--------------------------------------------------

A 层普通攻击切层最适合对接：
- Background Turret
- Sector Miner
- Cache Colossus
原因：
这些敌人需要 A 层先建立节奏，再进入 B 层兑现收益。

A 层炸弹切层最适合对接：
- Fragment Swarm
- Sector Miner
- 成组杂兵混战
原因：
这类敌人更看重区域控制和进场布置。

A 层控制切层最适合对接：
- Page-Lock Guard
- Background Turret
- Cache Colossus
原因：
玩家需要安全切入 B 层处理高压目标。

B 层组合切技最适合对接：
- Front Hound
- Anchor Lurker
- GC Enforcer
原因：
它们都适合单体转场和点杀收尾。

B 层重击切层最适合对接：
- Fragment Swarm
- Sector Miner
- Cache Colossus 的阶段战
原因：
它承担的是局面重排和群体带回，不是平替普通锤击。

B 层控制技切层最适合对接：
- Swap Priest
- Privilege Leech
- 需要快速回到 A 层指定位置的场面
原因：
它的价值主要在空间控制，不在硬伤害。

--------------------------------------------------
六、关卡投放顺序建议
--------------------------------------------------

第一批投放：
Front Hound
Page-Lock Guard
Background Turret

作用：
最先教会玩家“有些怪该送回 A，有些怪该去 B 杀”。

第二批投放：
Fragment Swarm
GC Enforcer
Sector Miner

作用：
开始测试群体带回、局面重排和稳杀 / 切回分流。

第三批投放：
Swap Priest
Privilege Leech
Anchor Lurker

作用：
让战斗从“只看单体怪层级”升级成“看战场优先级和空间风险”。

第四批投放：
Cache Colossus

作用：
作为第二章中后段对白盒战斗循环的综合检验。

--------------------------------------------------
七、淘汰标准
--------------------------------------------------

一只怪如果出现以下情况，就应该尽快删改，而不是硬留：

1. 不论在哪层打都差不多
2. 总被同一种招式无脑处理
3. 不能逼出任何切层判断
4. 只是在数值上更强，没有带来层级玩法差异
5. 混战里完全看不出它为什么存在

--------------------------------------------------
八、总结
--------------------------------------------------

第二章怪物设计的成功标准，不是怪物名字有多酷，而是：

1. 玩家能快速判断它在哪层更该处理
2. 玩家会自然做出“稳杀还是切回”的战术分流
3. 六种基础动作都能找到明确对接对象
4. 关卡混战时，双层系统会越来越有意思，而不是越来越乱

只要这四条成立，第二章的战斗生态就算真正搭起来了。
